Used Selenium IDE for FireFox Browser to run test automated test cases.

1. Install Browser FireFox, if not installed already
2. Install Selenium IDE by going to the following link: http://docs.seleniumhq.org/download/
3. Open FireFox Browser
4. Click on Selenium IDE icon at the top of Browser
5. Naviagate through File/AddTestCase and add html documents found in the tests folder
6. Play entire test suite or one test case at a time by clicking on one of the green buttons at the top
